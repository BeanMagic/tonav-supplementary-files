Detailed Description

template<typename T, typename Allocator = std::allocator<T>>
class BlockingDeque< T, Allocator >

Simple thread-safe blocking deque class.

All operations are thread-safe, but iterators are not. To use them in multiple threads, lock the object using the mutex().
